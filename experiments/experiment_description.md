Проведенный эксперимент воспроизводит алгоритм оценивания вероятности существования отношения пререквизита между заранее определенными понятиями, относящимися к разным областям знаний. Алгоритм был построен на основании общего описания, предложенного в статье (...), оттуда же был позаимствован датасет для сравнения результатов. Цель эксперимента состояла в том, чтобы реализовать вычислительную часть метода в неизменной логике, использовать ту же модель семантического кодирования и оценить, насколько получаемые значения метрики совпадают с эталонными значениями, приведёнными в исходном датасете.

Исходные данные представляли собой набор пар понятий и отдельный корпус текстовых описаний этих понятий. Для каждой пары в датасете было доступно эталонное значение метрики «csr_score», отражающей вероятность существования отношения пререквизита в этой паре по направлению от первого понятия ко второму. На первом этапе выполнялась унификация наименований понятий для корректного сопоставления записей из таблицы с файлами текстовых описаний. После этого для каждого понятия загружалось соответствующее текстовое описание, которое далее использовалось как основа для вычисления значения соответствующей метрики.

Ключевая часть эксперимента была связана именно с воспроизведением метрики CSR. Для этого текстовое описание каждого понятия разбивалось на последовательные блоки по десять слов. Далее название понятия кодировалось в векторное представление с помощью модели all-MiniLM-L6-v2. Тем же способом кодировались и все блоки описания другого понятия. После получения эмбеддингов выполнялась их нормализация, а затем вычислялась сумма косинусных расстояний между вектором названия понятия и всеми векторами блоков рассматриваемого описания. Полученное значение интерпретировалось как накопленная семантическая ссылка одного понятия на другое, то есть как значение CSR в заданном направлении.

Таким образом, для пары понятий A и B вычислялась направленная величина CSR(A→B), показывающая, насколько содержание текстового описания понятия A семантически соотносится с названием понятия B. В общей реализации метода для каждой пары понятий вычислялись обе направленные оценки, формировалась сводная величина ранжирования как максимум из двух направленных значений, а также учитывалась более простая метрика CER, отвечающая за прямые вхождения названия одной темы в описание другой. Помимо этого, значимая часть исходного алгоритма посвящена составлению минимального графа данных понятий, отражающего выявленные связи, однако эта часть подразумевает работу живого эксперта для однозначного определения существования отношения пререквизита между парой понятий, поэтому в данном эксперименте она не воспроизводилась. Внимание было сосредоточено именно на полностью автоматизированной части алгоритма, то есть на воспроизведении и верификации численных значений метрики CSR на эталонном наборе данных.

Из исходного набора данных выбиралась группа пар понятий, после чего для каждой такой пары заново вычислялось значение метрики CSR по описанной выше процедуре. Затем это значение сравнивалось с эталонным «csr_score», доступным в датасете, и вычислялось абсолютное отклонение. По совокупности пар для каждой проверяемой выборки рассчитывались среднее абсолютное отклонение и максимальное абсолютное отклонение. Такая постановка эксперимента позволяет оценить как общий уровень близости результатов, так и наличие единичных случаев с заметным расхождением.

Проверка была проведена на нескольких независимых фрагментах датасета по 100 пар понятий в каждом. Для первой выборки среднее абсолютное отклонение составило 0.746397, а максимальное — 13.171602. Для второй выборки среднее абсолютное отклонение составило 0.427687, максимальное — 2.327201. Для третьей выборки среднее абсолютное отклонение составило 0.491056, максимальное — 2.737585. В большинстве случаев вычисленные значения оказались очень близки к эталонным, что указывает на корректное воспроизведение общей логики расчёта, несмотря на единичные расхождения.

Полученные результаты позволяют сделать вывод о том, что реализованная процедура вычисления CSR в целом адекватно воспроизводит исходный метод. Небольшие средние значения абсолютной ошибки на проверенных подвыборках свидетельствуют о близости получаемых оценок к опубликованным значениям, и позволяют говорить о содержательно успешной реализации метода, а также свидетельствуют о работоспособности предложенного алгоритма, как инструмента определения вероятности существования отношения пререквизита, который может быть успешно применен в дальнейших этапах исследования.
